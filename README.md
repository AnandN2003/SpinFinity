# SpinFinity

Spinfinity is an online casino web application where players can enjoy the thrill of roulette in real-time. Built with the MERN stack, it offers seamless gameplay with live bidding, spinning, and instant updates using WebSockets. The platform supports two user roles: Players and Administrators.
The platform brings the thrill of roulette to life, allowing players to bid on chips, spin the roulette wheel, and experience the excitement of winning—all in an interactive, seamless environment. With a robust backend powered by WebSockets, real-time updates ensure players stay engaged without the need for page reloads.
